# ai-short-video-clip-assistant

AI短视频智能剪辑助手原型

项目经历：我曾主导一个名为“AI短视频智能剪辑助手”的课程项目。我利用ComfyUI搭建了从文生图、图生视频到自动配音的完整工作流，并通过调用Stable Diffusion和GPT-4 API，实现了根据热点话题一键生成短视频原型的功能。我重点优化了提示词工程，将视频素材的语义匹配准确率提升了约30%。该项目最终帮助模拟用户将单条视频的创作时间平均缩短了40%，并在校内展示中获得了最高用户满意度评分。

## 运行
```bash
pip install -r requirements.txt
python main.py
```
